# SNMP网络设备监控系统

## 项目简介
这是一个基于Python Flask和SNMP协议的网络设备监控系统，专门用于监控华为AR161W-S路由器(192.168.8.1)。系统提供实时Web界面，展示设备的各项性能指标。

## 功能特性

###  网络连通性监控
- 实时Ping测试，检测设备在线状态
- 响应时间监控
- 网络连通性状态指示

###  设备性能监控
- CPU瞬时使用率监控
- 内存瞬时使用率监控  
- 实时图表展示性能趋势

###  端口状态监控
- 所有网络接口状态监控
- 端口速率显示
- 进出流量统计
- 接口状态可视化

### ⏱️ 系统信息监控
- 设备运行时间(Uptime)
- 设备基本信息(型号、描述等)
- 系统联系人信息

###  网络流量监控
- 交换机播包出入传输速率
- 包传输速率图表展示
- 实时流量统计

## 技术架构

### 后端技术
- **Python 3.x** - 主要编程语言
- **Flask** - Web框架
- **PySNMP** - SNMP协议实现
- **ping3** - Ping测试库

### 前端技术
- **HTML5/CSS3** - 页面结构样式
- **JavaScript** - 动态交互
- **Chart.js** - 数据图表展示
- **响应式设计** - 支持移动端访问

### 监控协议
- **SNMP v2c** - 设备监控协议
- **ICMP Ping** - 网络连通性测试

## 安装部署

### 环境要求
- Python 3.7+
- Windows/Linux/macOS
- 网络访问权限(访问192.168.8.1)

### 快速开始

1. **克隆项目**
```bash
cd d:\AppData\VSCodeData\26-3
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **启动服务**
```bash
python run.py
```

4. **访问监控界面**
打开浏览器访问: http://localhost:5000

### 手动启动
```bash
# 方式1: 使用启动脚本(推荐)
python run.py

# 方式2: 直接启动Flask应用
python app.py
```

## 配置文件

### SNMP配置
- 设备IP: `192.168.8.1`
- SNMP团体字: `Public123`
- SNMP端口: `161`

### 监控配置
- 数据采集间隔: `30秒`
- 历史数据保留: `60个数据点`
- Web服务端口: `5000`

## API接口

### 获取当前数据
```http
GET /api/current_data
```
返回当前所有监控数据

### 获取历史数据
```http
GET /api/history_data
```
返回历史数据用于图表显示

### 获取设备信息
```http
GET /api/device_info
```
返回设备详细信息

### 获取接口信息
```http
GET /api/interfaces
```
返回网络接口状态信息

### Ping测试
```http
GET /api/ping_test
```
执行实时Ping测试

### 手动刷新
```http
GET /api/refresh
```
手动刷新监控数据

## 故障排除

### 常见问题

1. **SNMP连接失败**
   - 检查设备IP是否正确
   - 确认SNMP团体字配置
   - 验证网络连通性

2. **Web界面无法访问**
   - 检查5000端口是否被占用
   - 确认防火墙设置
   - 查看Flask服务日志

3. **数据不更新**
   - 检查监控线程是否正常运行
   - 查看SNMP查询权限
   - 验证设备SNMP服务状态

### 日志查看
监控过程中的错误信息会输出到控制台，便于问题诊断。

## 扩展开发

### 添加新的监控指标

2. 更新数据收集逻辑
3. 修改Web界面显示

### 支持更多设备类型
通过修改OID映射表，可以支持其他厂商的网络设备。
